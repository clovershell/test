# C/C++ 自动编译示例

## 概述
本项目包含多个 C/C++ 示例文件，并提供一个 Makefile，可自动扫描当前目录下的 `*.c` 与 `*.cpp` 源文件，生成同名可执行文件。

## 使用
- 编译全部：`make`
- 并行编译：`make -j`
- 清理可执行文件：`make clean`

## 说明
- 对于 `a.c` 会生成可执行文件 `a`，对于 `b.cpp` 会生成可执行文件 `b`
- 可通过命令行覆盖编译选项：例如 `make CFLAGS='-O0 -g'`

## 目录结构
- `Makefile`：自动编译规则
- `*.c / *.cpp`：示例源代码
